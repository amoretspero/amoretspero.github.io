AWS 백엔드를 기반으로 한 토양 품질 트래킹 시스템

Project Info:

이 프로젝트는 서울대학교 컴퓨터공학부의 2015년 가을학기 "창의적 통합 설계1" 과목의 일환으로, 학부생 3인이 AWS 멘토들과 함께 진행하였습니다.

본 시스템은 IoT 디바이스와 AWS의 여러 서비스를 통해 쉽게 구축할 수 있는 소규모 토양 품질 트래킹 시스템입니다. IoT 디바이스는 Intel Edison을 이용하였으며 이를 뒷받침하는 클라우드 서비스는 AWS의 Lambda, Kinesis, DynamoDB 등을 통해 구축되었습니다. 또한 함께 제공되는 앱을 통해 디바이스와 AWS 계정을 등록하면 해당 디바이스에서 트래킹하는 토양 데이터를 모바일 환경에서도 확인할 수 있도록 하였습니다.

제공되는 가이드를 통해 시스템 및 하드웨어 설정을 완료한 후에는 정해진 간격마다 토양 상태를 모니터링하여 정보를 클라우드에 기록하게 됩니다. 모바일 앱을 통해 사용자가 원하는 범위의 값을 설정하면 모니터링 정보가 범위를 벗어났을 경우 이메일로 알림을 보내므로 바로 옆에 있지 않더라도 쉬운 관리가 가능합니다.

  • 진행 기간: 2015/09 - 2015/12
  • 완료 여부: 완료
  • 기여도: 75%
  • 관련 기술: Arduino / AWS / Node.js
