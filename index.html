JavaBuild

Project description
This project is to provide simple command line based Java project build tool.
Developers who are familiar with make or eclipse may use them, but for most of just-started developers or who think writing make, turning on eclipse are very bothersome, JavaBuild provides simple JSON based build configurations, most of them are automatically managed by this program. JavaBuild is currently windows-only, and written with F#.
Language / Framework
F# / .NET Framework
Current Status
Build history management is provided, which will compile only updated source files.
User-specified source/class file location is supported.
Version
0.4.3.0 - Stable

Malloc Lab

Project description
Custom malloc function.
This malloc uses RB-tree implementation and heuristic base method for memory allocation. RB-tree is implemented based on description from wikipedia, and heuristic algorithm is to perform more efficiently with continuous allocation of repeated block size of specific sizes, while not hurting performance. Due to the fact that heuristic algorithm calculation and RB-Tree management is not optimized, speed may not be fast in resource-poor environments.
Language
C
