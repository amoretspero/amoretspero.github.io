자소서 바이블

Project Info:

자소서 바이블은 수험생들이 지원하는 학교/학과/전형에 맞게 자기소개서를 작성할 수 있도록 도와주는 서비스입니다.

본 서비스는 ASP.NET Core 프레임워크를 기반으로 하여 개발되었습니다. Server-side는 C#으로 구현되었으며 Client-side는 HTML/CSS 및 Javascript를 이용하여 구현되었습니다. 데이터베이스는 Azure SQL Database를, 스토리지 서비스는 Azure Blob Storage 서비스를 이용하고 있습니다. 또한 ASP.NET Core의 장점을 살려 현재 Azure Cloud Service의 Ubuntu Server를 운영체제로 하는 리눅스 서버에서 호스팅되고 있습니다.

자소서 바이블에서는 단순히 자기소개서를 작성하는 것 이외에 여러 기능을 이용할 수 있습니다. 학교, 학과 및 전형에 대한 정보와 가능한 경우 대학 입학처 및 학과 홈페이지 링크를 제공하여 수험생들이 자기소개서를 더 잘 쓸 수 있도록 돕고 있습니다. 또한 자기소개서 데이터베이스를 구축하여 자신이 지원하려는 대학, 학과, 전형에 지원했던 선배들이 썼던 자기소개서를 참고할 수 있도록 하며, 작성한 자기소개서는 인쇄하여 읽어보고 고칠 수 있도록 프린트 기능을 제공합니다.
